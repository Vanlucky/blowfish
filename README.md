该项目为heu 信息安全综合课程设计实验 blowfish算法实现（MFC框架）  
功能包括：  
1.blowfish加解密算法实现  
2.mfc友好页面  
3.文件操作、对话框操作  
4.密钥随机生成、清空内容、关于  

blowfish算法：
